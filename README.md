## 📖 About Agents

**_Agents_** is the version-controlled home for the rules, skills, subagents, slash commands, hooks, and custom MCP servers that drive the AI coding assistants in use on this machine: _Claude Code_, _OpenAI Codex CLI_, and _Gemini CLI_. The repository is self-installing — a single command on each supported operating system links the relevant files into each assistant's dotfile directory, so all three tools read from the same canonical source.

## 📚 Documentation

For detailed information on **_Agents_** installation and repository layout, please refer to the project's [Wiki](https://github.com/samariafar/agents/wiki). The Wiki provides comprehensive guides and resources to help you get started with **_Agents_** and make the most out of its features. Additionally, you can find answers to the most frequently asked questions (FAQs) in the Wiki.
